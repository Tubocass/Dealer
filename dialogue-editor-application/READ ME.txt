Dialogue Editor 0.9.1 
Creates and edits JSON fomatted dialogue graphs.

INSTALLATION INSTRUCTIONS
--------------------------------------------

You can either:
1. Place the contents of the Application Files directory in a folder of your 
choosing and access the program using a shortcut to Dialogue Editor.exe.
2. Run the DialogueEditor.application file in the Application Files directory 
to install the program on your system.

DOCUMENTATION
-------------------------------------

No documentation, as such, exists at this time. However, the program's 
GUI has many tooltips.
